Welcome to My School Project

A Simple Webpage Built with HTML & CSS

About This Project

This is my first webpage created for a school assignment. I learned how to use HTML for structure, CSS for styling, and how to deploy it using Git and Netlify. This project helped me understand the basics of web development and version control.

Fun Fact: This entire website is just one HTML file with embedded CSS. Simple but effective!

What I Learned

Through this project, I discovered how web technologies work together. HTML provides the structure and content, while CSS makes everything look good. I also learned about Git for version control and Netlify for hosting, which are tools that professional developers use every day.

Technologies Used

HTML5 - For the structure and content
CSS3 - For styling and making it look pretty
Git - For version control
Netlify - For hosting the website online
